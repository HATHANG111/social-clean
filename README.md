# Social Media App Backend

## Description
Backend for a Social Media App built with Next.js and TypeScript.  
Handles user authentication, posts, comments, and interactions.

## Tech Stack
- Next.js
- TypeScript
- TailwindCSS
- Prisma (database)

## Features
- User authentication (Sign Up / Sign In)
- User profiles
- Posts and comments
- Interactions (likes, shares)
